## Extension
- editorconfig
- vscode-icons

Générer l'editorconfig (right-click) puis le modifier

```
root = true

[*]
indent_style = space
indent_size = 2
end_of_line = crlf
charset = utf-8
trim_trailing_whitespace = true
insert_final_newline = true
```
